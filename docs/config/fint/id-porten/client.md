# Client

Configuration for **ID-Porten**.

This configuration is used by the Keycloak **ID-Porten** identity provider.
Some values may differ between environments.

## General

| Setting                         | Value / Guidance               |
| ------------------------------- | ------------------------------ |
| Client ID                       | Generated by ID-Porten         |
| Display name                    | `Flais IDP Fint <environment>` |
| Show organization name on login | Off                            |

## Properties

| Setting               | Value                |
| --------------------- | -------------------- |
| Application type      | `web`                |
| Authentication method | `client_secret_post` |


### Allowed grant types

| Grant type           | Value |
| -------------------- | ----- |
| `authorization_code` | On    |
| Refresh token        | Off   |

### SSO

| Setting                | Value |
| ---------------------- | ----- |
| Disable single-sign-on | On    |

## Lifetime & Tokens

| Setting                      | Value            |
| ---------------------------- | ---------------- |
| PKCE `code_challenge_method` | `S256`           |
| ID-token signing algorithm   | Standard `RS256` |
| Refresh token usage          | One-time         |
| Access token lifetime        | `120` seconds    |
| Refresh token lifetime       | `600` seconds    |
| Authorization lifetime       | `7200` seconds   |

## URI

### Redirect URI

```text
https://keycloak.<environment>.infra.flais.no/realms/<realm>/broker/<idp-alias>/endpoint
```

Example values:

| Placeholder     | Value       |
| --------------- | ----------- |
| `<environment>` | `prod`      |
| `<realm>`       | `fint`      |
| `<idp-alias>`   | `id-porten` |

### Post logout redirect URI

```text
https://keycloak.<environment>.infra.flais.no/realms/<realm>/broker/<idp-alias>/endpoint/logout_response
```
